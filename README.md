

## Project Overview

This project is a responsive gym business website built for local gyms that do not have an online presence. The website showcases services, gym information, and contact details to help potential customers easily connect with the business.

##🔍 Features

- Professional homepage design
- Services section
- About section
- Contact form
- Mobile responsive design
- Clean and modern UI
- Smooth navigation

## Pages Included

- Home Section
- Services Section
- About Section
- Contact Section

##🎯 Tech Stack

- HTML
- CSS
- JavaScript
